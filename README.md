# IoT-Based Smart Battery Management System using ESP32 and Blynk

## Project Overview
This project is an IoT-based Battery Management System developed using ESP32 and Blynk Cloud. It monitors multi-cell battery parameters, detects faults, provides protection, and displays real-time battery information.

## Features
- 4-cell voltage monitoring
- Pack voltage calculation
- Cell imbalance detection
- Battery health analysis
- Fault detection and protection
- Relay and buzzer control
- LCD-based user interface
- Blynk IoT cloud dashboard
- Real-time telemetry and diagnostics

## Technologies Used
- ESP32
- Arduino IDE
- Wokwi Simulator
- Blynk IoT Platform
- I2C LCD

## Implemented Tasks
- Task 1: Multi-cell Battery Intelligence Engine
- Task 2: Safety Protection Kernel
- Task 3: Embedded HMI Interface
- Task 4: Fault-Tolerant Runtime System
- Task 5: Cloud Telemetry Architecture
- Task 6: Executive Battery Intelligence Dashboard
